RAG Chatbot Project

Your project is served locally. This is the website view of the created RAG chatbot folder.

Open this page in your browser at:

http://localhost:8000

Files available in this folder:

  • app.py - chatbot runtime entrypoint
  • ingest.py - document ingestion and Chroma indexing
  • requirements.txt - dependencies
  • README.md - usage instructions
  • source_documents/example.txt - sample document

If you want to run the chatbot, open PowerShell in this folder and execute:

python ingest.py python app.py

Note: the current site is static and served from the local project folder.
